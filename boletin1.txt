1. Acceso remoto con SSH

ssh alumno08@dayhoff.inf.um.es

ssh -Y usuario@biomaster.atica.um.es

La opción -Y (o -X) habilita el reenvío de aplicaciones gráficas.

xeyes  # Aparece una ventana gráfica con unos ojos animados
gedit  # Abre el editor de texto gráfico
nautilus  # Abre el gestor de archivos gráfico

Para salir de la sesión remota: exit

2. Movimiento básico por el sistema

a) Usa una ruta absoluta para ir al directorio /usr/bin: 
cd /usr/bin
ls -l

b) Compara el contenido del directorio en terminal con un navegador gráfico:
nautilus /usr/bin  # Requiere un entorno gráfico

c) Regresa al directorio de inicio (home):
cd
pwd

d) Usa una ruta relativa para ir a /etc/X11 desde tu directorio home:
cd ../../etc/X11
pwd

3. Historial de comandos

a) Muestra el historial de comandos:
history

b) Accede a comandos previos:

Flechas de dirección ↑ ↓ para navegar por comandos.
Ctrl-R seguido de una palabra clave para buscar comandos.
Ejecuta un comando del historial usando su número: !25

4. Ayuda y documentación

a) Usa man para consultar opciones de un comando:
man ls

b) Usa info para obtener información detallada:
info ls

c) Visualiza documentación gráfica (requiere entorno gráfico):
konqueror man:/ls

5. Creación de directorios y archivos

a) Crea un directorio y archivos dentro de él:
mkdir ~/temp
cd ~/temp
touch archivo_vacio
mkdir dir2

b) Lista todo el contenido recursivamente:
ls -R
find .

c) Crea un archivo con contenido:
echo "Texto de ejemplo" > salida.txt

6. Metadatos de un fichero

a) Comprueba los metadatos de un archivo:
ls -l archivo_vacio

b) Usa du para ver el espacio ocupado por un directorio:
du -h ~/temp

c) Compara con un archivo en /usr/bin:
ls -l /usr/bin/ls

7. Consulta y modificación de permisos

a) Examina el archivo /usr/include/stdio.h con diferentes herramientas:
cat /usr/include/stdio.h
less /usr/include/stdio.h
kate /usr/include/stdio.h  # Editor gráfico
gedit /usr/include/stdio.h

b) Verifica los permisos y el propietario del archivo:
ls -l /usr/include/stdio.h

c) Explica los permisos del archivo, por ejemplo:
-rw-r--r--:
rw-: El propietario tiene permiso de lectura y escritura.
r--: Otros usuarios tienen permiso de lectura.
r--: El grupo también tiene permiso de lectura.

Compara con un archivo en /bin y uno en home:
ls -l /bin/ls
ls -l ~/temp/salida.txt
